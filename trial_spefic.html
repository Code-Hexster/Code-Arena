The Naming Spell

Every wizard's journey begins with a single spell — the Naming Spell. In the arcane language of Python, to name something is to create a variable.

A variable is like a magical chest: you give it a name, and it holds a value inside. Today, you will cast your first spell.

Objective Parameter

Create a variable called hero_name and set it to a string containing your wizard hero's name. The program will print a welcome message.

Hint 1
To declare a string in Python, enclose your text within quotes. For example, "Gandalf" or 'Merlin'.
Hint 2
Make sure you spell the variable exactly as requested: hero_name. Variable names in Python are case-sensitive.
